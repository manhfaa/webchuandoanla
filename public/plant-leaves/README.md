Plant leaf image sources
=========================

This folder stores local plant leaf images used by the public landing page.
Images are sourced from Wikimedia Commons. Keep source URLs and credits in
`src/data/mock/plants.ts` when adding or replacing assets.

- apple.jpg: Wikimedia Commons / Biswarup Ganguly
  https://commons.wikimedia.org/wiki/File:Malus_domestica_-_Apple_Leaves_-_Kullu_-_2014-05-09_2200.JPG
- blueberry.jpg: Wikimedia Commons / Derek Ramsey
  https://commons.wikimedia.org/wiki/File:Highbush_Blueberry_Vaccinium_corymbosum_Leaves_3008px.jpg

The remaining cards currently load from `upload.wikimedia.org` with visible
source links in the UI because direct batch download was temporarily rate
limited by Wikimedia.
